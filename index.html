Project Xanadu и WWW

Xanadu - первая концепция гипертекста, созданной программистом Тедом Нельсоном в 1960 году

Реализация вышла 1998. Немного поздновато.

Project Xanadu - 17 правил

  1. Каждый пользователь может искать, скачивать, создавать и хранить документы.
  2. Каждый документ может состоять из любого количества частей, каждая из которых может быть данными любого типа.
  3. Каждый документ может содержать ссылки любого типа, включая виртуальные копии ("включения") любого другого документа в системе, доступного владельцу.
  4. Каждый документ может содержать механизм вознаграждения автора с любой степенью подробности для обеспечения выплаты по частичному чтению документа, в том числе и для ("включений") любой части документа.
  5. Каждый документ уникальным образом безопасно идентифицируется.
  6. Каждый документ может иметь защиту для контроля доступа.
  7. Каждый документ может быть найден быстро, храниться и скачиваться без знания пользователем физического положения документа.
  8. Каждый документ автоматически перемещается в физическое хранилище, соответствующее частоте доступа к нему из любой данной точки.
  9. Каждый документ автоматически сохраняется с избыточностью, позволяющей сохранить доступ к нему даже в случае катастрофы.
  10. Каждая транзакция безопасна и поддаётся проверке только теми лицами, которые её совершают.
  11. Протокол коммуникации клиент-сервер Xanadu - открыто опубликованый стандарт. Разработка и интеграция программного обеспечения сторонними разработчиками поощряется.
